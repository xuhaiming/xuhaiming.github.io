As the propose of this website is writing tech blog, it is very important to give a good view of example codes. "pre" and "code" tags provide the developers possibility to use HTML on their website, but the styling and keyword highlight still need effort to do.

Fortunately, there is a greatlibrary called Highlight JS can help us achieve this goal. Just few lines of code following the Highlight JS guide, you will be able to customize your code in the website.

Although it is possible to use NPM to install the Highlight JS package, I still prefer to directly reference the JavaScript file because I can customize what I need to support in this library. 

In the component did mount function on my blog jsx file, I initialized the Highlight JS function:

    
        hljs.initHighlightingOnLoad();

After the HTML has been loaded to the DOM, I applied all texts in "pre" "code" tags with the seleted style:

    
        $.get(blogUrl + ".html", (htmlContent) => {
            this.setState({htmlContent: htmlContent}, () => {
                $('pre code').each((i, block) => {
                    hljs.highlightBlock(block);
                });
            });
        });

Then it works! Really simple, isn't it?
